## Local Development

**Install dependencies**

\`\`\`bash
pnpm i
\`\`\`

**Add your BASEHUB_TOKEN to `.env.local`**

\`\`\`txt
# .env.local

BASEHUB_TOKEN="<get-it-from-your-basehub-repo>"
RESEND_API_KEY="" # get it here https://resend.com/
\`\`\`

**Start the dev server**

\`\`\`bash
pnpm dev
\`\`\`
